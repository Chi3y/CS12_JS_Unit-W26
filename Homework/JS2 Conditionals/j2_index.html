Instructions

For the following problems you will write programs using the prompt() and will also use variables. Please have ALL OUTPUT to ALERT. Practice good string formatting.

Question 1

Grade_Boundaries Create a grading program that enables a student to enter their grade on an assignment. Depending on the grade, the program should display a message similar to below.

  • 90-100: "Excellent!"
  • 80-89: "Great Work!"
  • 70-79: "Well Done!"
  • 60-69: "Pass!"
  • 0-59: "Fail!"

Question 2

Recall_Number There have been some safety recalls on a number of cars at your dealership.

Cars with model numbers 119, 179, 189 through 203, and 235-252 are all found to be defective. Create a program to enable the user to check the model number. Prompt for model number, print “Recalled” or “Not recalled”. ** Consider all number ranges as inclusive.

Question 3

Weekly_Pay Create a wages program for your employees to calculate weekly pay. Prompt for the employee name, rate of pay, and number of hours. If the employee works more than 40 hours, they will be paid an overtime rate of 1.5 x normal hourly rate for any hours over 40.

Example:

Name = Wendy

Rate of Pay = $20

Number of hours = 45

Calculate: Pay of Wendy = $20 x 40 + $20 x 1.5 x 5 = 800 + 150 = $950

Tests for Q3
  • hours = 30, rate = 10 -> Pay = $300
  • hours = 40, rate = 20 -> Pay = $800
  • hours = 45, rate = 20 -> Pay = $950
